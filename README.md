# Digital-Library-Book-Management-System
## 🚀 Features
- Add, View, Update, Delete Books
- Search by ID or Title
- API Documentation with Swagger

## 🛠️ Setup Instructions
1. Clone the repo  
2. `cd book-catalog`  
3. `mvn clean install`  
4. `java -jar target/book-catalog-0.0.1-SNAPSHOT.jar`  

## 🚀 API Endpoints
- POST `/api/books`
- GET `/api/books`
- PUT `/api/books/{id}`
- DELETE `/api/books/{id}`
